# Conversion

Simple PHP unit conversion tool.

## Usage

Run in command line

```
php convert.php <value> <unitFrom> <unitTo>

// Ex: php convert.php 2 kg oz
// Outputs: 2.00 kg is 70.54 oz
```

### Supported units

#### Weight

- g (gram)
- kg (killogram)
- mg (milligram)
- ug (microgram)
- oz (ounce)
- lb (pound)

#### Temperature

- C (Celsius)
- K (Kelvin)
- F (Fahrenheit)
